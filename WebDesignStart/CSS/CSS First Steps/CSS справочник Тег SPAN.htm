Использование тега SPAN

Вопреки распространенному мнению тег <span> входит в стандарт HTML 4.0 (не верите - проверьте на http://www.w3.org/TR/1998/REC-html40-19980424/). Этот тег специально зарезервирован в HTML для использования бестеговых стилей. По умолчанию тег span ничего не делает с находящимся в нем текстом. Предположим, что некоторые имена на сайте вы хотите выделить синим цветом, причем вы не хотите выделять их ни жирным, ни наклонным шрифтом, короче вы не хотите привязывать это выделение ни к одному тегу. Тогда тег <span> - идеальное решение. Определите следующее стилевое правило: 


span.blue{color: blue}

а потом выделяйте текст следующем образом: 


<span class="blue">blue text</span>
