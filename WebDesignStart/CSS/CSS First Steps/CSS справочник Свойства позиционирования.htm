Система координат

Как и в языках программирования, система координат в модели CSS отличается от декартовой. Начало координат находится в левом верхнем углу окна. Координата x возрастает слева на право, а координата y - сверху вниз.


Свойство left

Это свойство определяет x-координату верхнего правого угла элемента. Значение может задаваться либо длиной, либо в процентах от ширины окна броузера. Может также принимать значение auto, указывающие броузеру, что надо выбрать x-координату элемента автоматически.

Свойство top

Свойство top задает y-координату верхнего правого угла элемента. Значение может задаваться либо длиной, либо в процентах от высоты окна броузера. Как и left может принимать значение auto.

Свойство position

Это свойство указывает броузеру относительно какого элемента располагать содержимое тега в окне. Может принимать четыре значения: absolute (координаты задаются относительно верхнего левого угла родительского элемента), static(координаты задаются относительно верхнего левого угла страницы), relative (координаты задаются относительно предыдущего элемента) и fixed (координаты задаются относительно левого верхнего угла окна браузера клиента). Однако стоит отметить, что значение fixed работает только в последних версиях NN и Opera.

Свойство z-index

Свойство z-index задает порядок перекрытия элементов друг друга. Если элементы перекрываются, то сверху будет отображаться элемент, имеющий большее значение z-index. Этот параметр может задаваться либо числом, либо ключевым словом auto, которое меньше любого значения z-index. Если этому свойству двух элементов присвоено значение auto и они перекрываются, то отображаться сверху будет элемент, определенный позже в исходном коде. Значение auto принято по умолчанию.
