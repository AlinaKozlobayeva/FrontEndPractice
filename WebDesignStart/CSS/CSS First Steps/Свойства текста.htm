Свойство letter-spacing

Это свойство определяет дополнительные интервалы между буквами. Значение свойства может быть длинной(в пикселях, пунктах, процентах и т.д.) или ключевым словом normal.

Свойство line-height

Это свойство указывает высоту строки для определения расстояния между строками текста. Значением этого свойства может быть абсолютная или относительная длина.

Свойство text-align

Это свойство определяет выравнивание текста относительно краев страницы. Может принимать одно из значений: left(по левому краю), right(по правому краю), center(по центру) и justify(по левому и по правому краю одновременно, раздвигая слова и символы по мере необходимости). По умолчанию это свойство принимает значение left.

Свойство text-decoration

Создает украшение текста, может принимать одно из значений: underline( подчеркивание), overline (надчеркивание), line-through (зачеркивание), blink (мерцание). Значение blink, как и тег <blink>, интерпретируется только броузерами Netscape Navigator.

Свойство text-indent

Формирует отступ первой строки элемента(в основном абзаца). Может принимать абсолютное или относительное значение. По умолчанию принимает значение 0. Применяйте это свойство для создания красной строки: 


{text-indent: 3em}

Свойство text-shadow

Это свойство позволяет принять тексту трехмерный вид. Значение этого свойства - четыре параметра, разделенные пробелом. Первый два - сдвиг тени, далее необязательный параметр радиус, затем - цвет тени. Пример: 


h1{text-shadow: 10px 10px 2px red}.

Ни один из популярных броузеров не поддерживает это свойство, так что не будем долго на нем останавливаться

Свойство vertical-align

Это свойство указывает, как располагать данный элемент по отношению к содержащей его строке. Принимает следующие значения:

baseline
Выравнивает элемент по базовой линии строки, содержащий этот элемент.
middle
Выравнивает середину элемента, с серединой охватывающего текста(обычно с верхом строчной буквы)
sub
Выводит элемент как подстрочный индекс
supper
Выводит элемент как надстрочный индекс
text-top
Выравнивает верхний край элемента по верхнему краю охватывающего текста
bottom
Выравнивает нижний край элемента по нижнему краю охватывающего текста

Netscape и Explorer лишь частично поддерживают эти свойства.

Свойство word-spacing

Используется для увеличения интервала между словами. Может принимать значение normal или значение длины. Только Netscape Navigator 6 и далее поддерживает это свойство.

Свойство text-transform

Это свойство позволяет преобразовать содержимое тега. Может принимать значения: capitalize(выводит первую букву каждого слова в верхнем регистре), uppercase (выводит все в верхнем регистре), lowercase (выводит все в нижнем регистре), none.
