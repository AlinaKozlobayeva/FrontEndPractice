Где работают эти свойства

Все свойства, описанные в этом разделе задают цвет отображения разных частей полосы прокрутки (или scrollbar) и начинаются с префикса scrollbar. Все эти свойства принимают стандартные цветовые значения. Внимание: эти свойства не входят в стандарт CSS2, они будут работать лишь в браузрах Microsoft Internet Explorer начиная с версии 5.5.

Свойство scrollbar-base-color

Это свойство задает основной цвет полосы прокрутки. Если не определены другие свойства полосы прокрутки, то бегунок и фон кнопки прокрутки будут отображаться определенным вами цветом с применением трехмерных эффектов. Фон полосы прокрутки будет отображаться тоже этим цветом, но только осветленным.

Свойство scrollbar-face-color

Это свойство определяет цвет бегунка и фона кнопки прокрутки

Свойство scrollbar-track-color

С помощью свойства scrollbar-track-color можно определить фон полосы прокрутки.

Свойство scrollbar-shadow-color

Internet Explorer применяет трехмерные эффекты при рисовании бегунка и кнопок прокрутки. По правой и нижней границе он проводит две линии вдоль границы толщиной в один пиксель. Первая с внутренней стороны линяя имеет цвет, определенный свойством scrollbar-shadow-color, вторая цвет, определенный свойством scrollbar-darkshadow-color. Соответственно по левой и верхней границе тоже проводятся две линии, которые имеют цвета совпадающие со значениями свойств scroll-bar-3dlight-color и scroll-bar-hightlight-color идя с внутренней стороны. Свойство scrollbar-highlight-color

Свойство scrollbar-arrow-color

Как вы уже, наверное, догадались, это свойство задает цвет изображения стрелки на кнопке прокрутки.
