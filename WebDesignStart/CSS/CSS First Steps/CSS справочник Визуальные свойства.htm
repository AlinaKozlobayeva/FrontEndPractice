Свойство cursor

Свойство cursor определяет вид курсора над элементом.Может принимать следующие значения (если ваш броузер поддерживает стили, то, поместив указатель мыши над соответствующим названием, вы увидите соответствующий курсор):crosshair, hand, move, e-resize, ne-resize, nw-resize, n-resize, sw-resize, se-resize, s-resize, w-resize, text, wait, help, auto(стандартная форма курсора для данного элемента), default(стандартная форма курсора для окна броузера).

Свойство visiblity

Это свойство управляет видимостью элемента. Может принимать три значения: inherit(значения данного свойства наследуется от родительского элемента), none(элемент невидим в том смысле, что вместо него отображается пустая область тех же размеров, что и сам элемент), visible(элемент видим).

Свойство display

Свойство display способом управляет способом отображения элемента. Это свойство может принимать такие значения: block (элемент отображается как блочный; к блочным элементам относятся таблицы, абзацы, заголовки и т.д.), inline (элемент отображается как встроенный; к встроенным элементам относятся теги физической и логической разметок, гиперссылки и т.д.), list-item (элемент отображается как элемент списка и помечается маркером), none (элемент не отображается вместе со всем своим потомством, предыдущий и последующий элементы сдвигается вместе).

Свойство overflow

Это свойство задает стиль отображения текста, переполневшего границы элемента. Может принимать следующие значения: scroll (создать линейки прокрутки для просмотра не влезшего содержимого), hidden (спрятать содержимое, не попавшее в видимую часть элемента), visible (все содержимое тега будет видимо, даже если заданы свойства height и width); в этом случае броузер "растянет" элемент), inherit (значение наследуется у родительского элемента), auto (броузер сам решит, что делать с текстом, переполнившим границы элемента).

С помощью этого свойства можно создать некое подобие фреймов. Оглавление будет располагаться всегда справа, о содержание документа можно будет прокручивать. Это можно сделать так:

<head>
<title>Заголовок</title>
<style>
div{height: 100%; overflow:scroll;} /*содержание тега div, будет
прокручиваться*/
</style> 
</head>
<body>
<table width="100%" height="100%">
<tr>
<td width="20%" valign="top">
