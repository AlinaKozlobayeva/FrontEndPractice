Свойство list-style-type

Свойство list-style-type управляет типом вывода как упорядоченных, так и неупорядоченных списков. Оно аналогично атрибутом type тегов <ol> и <ul>.

Для элементов неупорядоченных списков это свойство определяет форму маркеров и принимает следующие значения: disc (диск), circle (окружность), square (квадрат) или none (нет маркера). По умолчанию принимается значение disc; броузеры изменяют принятое по умолчанию значение в зависимости от глубины вложенности списка.

Свойство list-style-type, примененное к элементу упорядоченного списку, определяет вид нумерации. Может принимать следующие значения: decimal(арабские цифры), lower-roman(маленькие римские цифры), upper-roman(большие римские цифры), lower-alpha(строчные латинские буквы), upper-alpha (заглавные латинские буквы) и none(нет нумерации). Значения по умолчанию стандартами CSS не определено.

Свойство list-style-image

Это свойство определяет изображение, используемое вместо маркера. Укажите в значении свойства URL изображения, и вместо обычного маркера(если, конечно, удалось загрузить изображение) броузер выведет ваш рисунок. Пример: li {list-style-image: url(img/marker1.gif)}.

Свойство list-style-position

Свойство list-style-position управляет способом вывода маркера. Может принимать два значения: inside(маркер расположен снаружи) и outside(маркер включен в текстовый блок). Ни один из популярных броузеров не поддерживает это свойство.

Область применения свойств списков

Вообще свойства списков могут быть применены к любым элементам, свойство display которых имеет значение list-item. По умолчанию единственный тег, который обладает этим свойством, - это тег <li>;. Однако вы можете применять эти свойства и тегам <ol> и <ul>. В этом случае определенные свойства перейдут на теги <li>, которые находятся внутри тегов <ol> и <ul>.
