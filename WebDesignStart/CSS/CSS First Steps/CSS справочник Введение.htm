Что такое CSS?

CSS - расшифровывается как Cascading Style Sheets, или по-русски каскадные таблицы стилей. Вообще таблицы стилей это не изобретение веб-дизайнеров, они давно используются в издательских системах для управления внешним видом документов. Так и в HTML таблицы стилей управляют внешним видом документа (на самом деле задачи CSS намного шире, они могут управлять отображением не только на визуальные устройства, но и, например, синтезаторы речи, но пока ни один из популярных броузеров не поддерживает эти возможности CSS).

Поясню на примере: вы хотите, чтобы каждое выделенное жирным шрифтом слово подсвечивалось еще и красным цветом. Вы, конечно, можете добиться нужного эффекта, выделяя каждое слово тегом <font color="red">, однако это очень громоздкая конструкция, с помощью стилей этого можно достичь проще. Вы можете определить стиль тега <b> (в каком месте это нужно сделать вы узнаете позже) следующим образом: 


b {color: red}

тогда все, что находится внутри тега <b>, будет выделено красным цветом. Такой способ намного проще предыдущего способа, к тому же измениние цвета делается в одно касание измененением стиля.

Рамки этого сайта

Этот сайт ни в коей мере не претендует на полное руководство по CSS. В нем лишь описываются основные свойства, поддерживаемые популярными броузерами. Если вы хотите изучить полное руководство по CSS2, перейдите в раздел ссылки, там вы найдете спецификацию CSS2 и ее русский перевод.
