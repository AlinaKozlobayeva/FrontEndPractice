Свойство font-family

Значение этого свойства - список шрифтов, которыми броузер должен отобразить данный объект или одно из семейств шрифтов, определенных стандартом CSS, такие как: serif (с засечками), sans-serif (без засечек), fantasy (с украшениями) и monospace (моноширинные). Если указан список шрифтов, то броузер выберет первый в списке, который есть на компьютере пользователя, если указано семейство, то броузер сам решает, какой шрифт из этого семейства выбрать. 


h1 {font-family: Helvetica, sans-serif}

Если на компьютере пользователя есть шрифт Helvetica, то заголовки будут отображаться именно этим шрифтом, а если такого шрифта нет, то броузер будет отображать их шрифтом семейства sans-serif.

Свойство font-size

Это свойство определяет размер шрифта. Может принимать одно из стандартных значений размера или одно из ключевых слов для абсолютного размера. Вот эти ключевые слова по порядку увеличения шрифта: xx-small, x-small, small, medium, large, x-large, xx-large. Эти размеры обычно соответствуют семи размерам шрифта, используемым для атрибута size тега <font> (за описанием этого тега загляните в НTML справочник).

Свойство font-stretch

Значение этого свойства указывает, насколько сильно надо растягивать буквы шрифта. Может принимать следующие значения, начиная с самого плотного: ultra-condenced, extra-condenced, condenced, semi-condenced, semi-expanded, expanded, extra-expanded, ultra-expanded. Популярные броузеры не поддерживают это свойство.

Свойство font-style

Может принимать три значения: normal(нормальное начертание), italic(курсивное начертание), oblique(наклонное начертание). Курсивное и наклонное начертании обычно не различаются.

Свойство font-variant

Может принимать два значения: normal, small-caps. Значение small-caps указывает броузеру, что надо заменить строчные буквы на маленькие заглавные.

Свойство font-weight

Используется для управления жирностью шрифта. Может принимать значение от 100(самый тонкий) до 900(самый жирный). Значение 400 можно заменить ключевым словом bold, а значение 700 - ключевым словом bold. Также значением этого свойства могут быть ключевые слова lighter и bolder, которые показывают, что надо использовать версию шрифта, которая соответственно менее или более жирная, чем принятая для родительского элемента

Краткое описание свойств шрифта

Стилевое правило: 


div
{
   font-family: Arial, "Comic Sans MS";
   font-weight: bold;
   font-size: 12px;
   line-height: 14px
;}
эквивалентно следующему (см. раздел Значение свойств): 

div{font: bold 12px/14px Arial, "Comic Sans MS"}
Порядок перечисления играет существенное значение. Сначало идет значение font-weight, потом - размер шрифта и через слеш его высота, и лишь затем значение font-family.
